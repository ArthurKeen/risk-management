# Project Presentation

The project overview is available as a PDF in this folder.

**[Sentries Overview](sentries-overview.pdf)** — project overview (add `sentries-overview.pdf` to the `docs/` folder after exporting from your slides).

---

## How to add or update the presentation

1. Export your presentation from Google Slides (or another tool): **File → Download → PDF**.
2. Save the file as `docs/sentries-overview.pdf` in this repo.
3. Commit the file. Re-export and replace the file whenever you update the slides.

The PDF is versioned with the repo and works offline.
